## üöÄ How to Use Profession-Specific Questions

### Automatic Role Detection

The system can automatically select questions based on role keywords in job description:

```python
retriever = QuestionRetriever()
session = InterviewSession()

# Medical interview
questions = retriever.retrieve_phased(
    session=session,
    resume_text="MBBS graduate with internship experience",
    job_description="Medical Officer position at City Hospital",
    top_k=10
)
# Will prioritize medical_professional questions

# Aviation interview
questions = retriever.retrieve_phased(
    session=session,
    resume_text="Commercial pilot license holder",
    job_description="First Officer position at airline",
    top_k=10
)
# Will prioritize pilot_aviation questions
```

### Manual Role Selection

Or explicitly filter by role:

```python
# Get only medical questions
medical_qs = [q for q in retriever.medical_questions if q['id'] not in session.asked_questions]

# Get only law questions
legal_qs = [q for q in retriever.lawyer_questions if q['id'] not in session.asked_questions]
```
